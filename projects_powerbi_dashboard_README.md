# Madhav E-Commerce Sales Dashboard

## 📊 Project Overview

Built an interactive Power BI dashboard for Madhav E-Commerce business that tracks key performance indicators (KPIs) across sales, orders, and product categories. The dashboard enables quick identification of top-performing segments, sales trends, and areas for growth.

## 🎯 Business Problem

Madhav E-Commerce needed a centralized, real-time view of:
- Total revenue and order trends
- Which product categories drive the most profit
- Customer purchasing patterns
- Month-over-month sales performance
- Regional/segment performance

Without a dashboard, stakeholders had to manually compile reports, leading to delays and missed insights.

## 📊 Dashboard Metrics & KPIs

✅ **Revenue Tracking** — Total sales, revenue by category, monthly trends

✅ **Order Analytics** — Order count, order frequency, average order value

✅ **Product Performance** — Top-performing categories, profit margins, inventory insights

✅ **Sales Trends** — YoY growth, seasonal patterns, forecasting

✅ **Customer Insights** — Purchase behavior, repeat customer rate, regional preferences

## 🛠️ Tools & Techniques Used

- **Tool:** Microsoft Power BI Desktop
- **Techniques:**
  - Data modeling and relationship building
  - DAX measures and calculated columns
  - Interactive filters and slicers
  - Conditional formatting for visual alerts
  - Multi-page dashboard design
